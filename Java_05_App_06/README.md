# 성적처리 프로젝트
* 학생의 국,영,수 점수를 저장 할 model.ScoreVO 클래스 작성
* service.ScoreService 클래스를 작성
* List<ScoreVO> 리스트 선언
* makeScore() method에서 각 과목에 50~100까지 임의의 정수를 생성하여 저장하고 List를 20개 작성
* scorePrint() method에서 성적 리스트 출력
```
=============================================
학번	국어	영어	수학	총점	평균
---------------------------------------------
1	90	90	90	270	90.0
=============================================
총점	000		000		000		000
```
